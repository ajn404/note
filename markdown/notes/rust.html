# 位运算

运算符说明
& 位与相同位置均为1时则为1，否则为0
| 位或相同位置只要有1时则为1，否则为0
^ 异或相同位置不相同则1，相同则为0
! 位非把位中的0和1相互取反，即0置为1，1置为0
<< 左移所有位向左移动指定位数，右位补0
>> 右移所有位向右移动指定位数，带符号移动（正数补0，负数补1

    let s = "holla中国人नमस्ते";
    // for item in s.bytes(){
    //     println!("{}",item);
    // }
    
    for item in s.chars() {
        println!("{}", item);

    let sub_s = utf8_slice::slice(s, 2, s.len() - 2);

}
